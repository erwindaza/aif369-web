# Entorno Profesional para Desarrollar IA con Agentes

## Descripción breve
Configura tu máquina para trabajar como AI Engineer desde día 1. Este curso te guía a través de las herramientas profesionales que todos los desarrolladores de IA usan: control de versiones, entornos aislados, secretos seguros, y tu primer agente funcional.

**No es un curso de instalación.** Es un curso de *flujo de trabajo profesional*: cómo mantener el código limpio, seguro y reproducible mientras trabajas con APIs de IA.

## ¿Qué aprenderás?

- ✅ Configurar VS Code como un entorno completo de desarrollo (extensiones, debugging, terminal integrada)
- ✅ Usar Git y GitHub desde la terminal como un profesional (commits, ramas, pull requests)
- ✅ Gestionar dependencias de Python de forma aislada y reproducible con `uv`
- ✅ Contenerizar tu código con Docker (primeros pasos, sin necesidad de ser un experto)
- ✅ Manejar secretos y variables de entorno sin exponerlos en GitHub
- ✅ Conectarte a OpenAI, Google Gemini, Anthropic Claude desde Python
- ✅ Asistentes de programación: Codex, Claude Code, Gemini CLI (cuándo usar cada uno)
- ✅ Construir y ejecutar tu primer agente de IA que tome decisiones

## Requisitos previos

- Una computadora (Windows, Mac o Linux) con permisos de administrador
- Curiosidad por la IA y disposición a escribir código
- Experiencia básica con terminal es útil, pero no obligatoria — te lo explicamos todo
- Coneccción a Internet estable

## No necesitas:
- Ser experto en programación
- Tener experiencia previa con IA
- Saber Docker de memoria — lo enseñamos desde cero

## Duración
- **2 a 3 horas** de contenido en video + ejercicios
- Puedes verlo en 1-2 sesiones o por partes

## Certificado
Al completar todos los ejercicios y validaciones, recibirás el **Certificado de Entorno Profesional AIF369**.

## Lo que construirás
Un repositorio en GitHub con:
- Entorno aislado y reproducible con `uv`
- Archivo `.env` securizado (nunca en GitHub)
- Dockerfile funcional
- Tu primer agente que se conecta a una API de IA y toma decisiones

Podrás mostrar este repo en tu portafolio profesional.
