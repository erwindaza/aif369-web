Ejemplos Reales

Aprende con proyectos reales

Estos son ejemplos de proyectos que se construyen durante los cursos de AIF369. Soluciones que puedes replicar, adaptar y llevar a tu trabajo.

Pipeline de datos

Pipeline ETL con Airflow + BigQuery

Ingestión de datos desde una API pública, transformación con pandas y carga a BigQuery. Orquestado con Airflow, ejecutándose diariamente de forma automática.

  • Airflow DAG con 5 tareas encadenadas
  • Extracción de API REST con paginación
  • Transformación y validación de datos
  • Carga incremental a BigQuery

Se enseña en: Automatización con Airflow

IA + Clasificación

Clasificador de textos de soporte

Modelo de IA que clasifica automáticamente tickets de soporte técnico en categorías, prioridades y asigna al equipo correcto.

  • Fine-tuning de modelo de lenguaje
  • API REST para clasificación en tiempo real
  • Integración con sistema de tickets
  • Dashboard de métricas del modelo

Se enseña en: Curso de IA + Airflow + IA Avanzada

Dashboard

Dashboard automatizado con datos en tiempo real

Dashboard ejecutivo que se actualiza automáticamente cada hora con métricas de negocio, KPIs y alertas configurables.

  • Ingestión de datos de múltiples fuentes
  • Transformación y agregación automática
  • Visualización con Chart.js/Looker
  • Alertas por email y Slack

Se enseña en: Big Data + IA

IA Generativa

Bot de atención con IA generativa

Chatbot basado en RAG que responde preguntas sobre documentación interna de la empresa usando IA generativa con contexto real.

  • Ingestión de documentos (PDF, DOCX, web)
  • Embeddings y vector store
  • RAG pipeline con Gemini/GPT
  • Chat widget integrado en web

Se enseña en: Airflow + IA Avanzada

Automatización

Monitoreo de costos cloud con IA

Sistema que monitorea costos de GCP/AWS en tiempo real, detecta anomalías con ML y envía alertas antes de exceder presupuestos.

  • BigQuery billing export integration
  • Anomaly detection con ML
  • Dashboard de costos por servicio
  • Alertas proactivas por presupuesto

Se enseña en: Big Data + IA + Airflow

Data Quality

Validación inteligente de datos

Pipeline que valida datos entrantes usando reglas estáticas + IA para detectar anomalías semánticas y patrones fuera de lo normal.

  • Great Expectations integrado en Airflow
  • Validación semántica con LLMs
  • Quarantine de registros anómalos
  • Reportes automáticos de calidad

Se enseña en: Airflow + IA Avanzada
