📊 ¿De qué trata este programa?

Un programa intermedio que conecta el poder de Big Data con Inteligencia Artificial. Aprenderás a procesar, analizar y enriquecer datos masivos con técnicas de IA.

Diseñado para profesionales con experiencia en datos que quieren dar el salto hacia arquitecturas modernas de Big Data con componentes inteligentes.

Requisito: conocimientos básicos de SQL, Python y conceptos de bases de datos.

Detalles del programa

  • Nivel: Intermedio
  • Duración: 8 semanas (16 horas)
  • Modalidad: Online en vivo + grabaciones
  • Sesiones: 2 horas por semana
  • Proyecto final: Pipeline Big Data + IA
  • Certificado: AIF369 Big Data + IA

Contenido del programa

8 módulos con arquitecturas, herramientas y proyectos reales.

Módulo 1: Fundamentos de Big Data

  • Qué es Big Data: volumen, velocidad, variedad
  • Data lakes vs data warehouses
  • Arquitecturas modernas: lakehouse, medallion
  • Ecosistema de herramientas Big Data

Módulo 2: Procesamiento de datos masivos

  • Apache Spark fundamentals
  • BigQuery y procesamiento en la nube
  • ETL vs ELT: cuándo usar cada uno
  • Práctica: procesar 1M+ registros

Módulo 3: Data Engineering moderno

  • Pipelines de datos en producción
  • Calidad de datos a escala
  • Orquestación básica de procesos
  • Monitoreo y alertas de pipelines

Módulo 4: IA sobre Big Data

  • Machine Learning en datos masivos
  • Feature engineering a escala
  • Entrenamiento distribuido
  • Caso: modelo predictivo con millones de registros

Módulo 5: IA Generativa + Datos

  • RAG (Retrieval-Augmented Generation)
  • Embeddings sobre datos empresariales
  • Búsqueda semántica en documentos
  • Caso: chatbot con datos internos

Módulo 6: Arquitectura en la nube

  • GCP: BigQuery, Cloud Storage, Vertex AI
  • AWS: S3, Redshift, SageMaker
  • Azure: Synapse, ADLS, Azure ML
  • Costos y optimización de recursos

Módulo 7: Gobernanza de datos

  • Catálogo de datos y metadata
  • Linaje de datos (data lineage)
  • Privacidad y cumplimiento regulatorio
  • IAM y seguridad en datos

Módulo 8: Proyecto final

  • Pipeline completo: ingestión → procesamiento → IA
  • Despliegue en nube real
  • Presentación ejecutiva y técnica
  • Feedback del instructor + portfolio
