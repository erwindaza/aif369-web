IA y Automatización de Datos con Airflow

Orquesta procesos de datos con Apache Airflow e integra IA en pipelines modernos de producción.

🎓 Precio especial de lanzamiento

⚡ ¿De qué trata este curso?

Apache Airflow es el estándar de la industria para orquestación de datos. Este curso te enseña a construir, programar y monitorear pipelines de datos con Airflow desde cero.

Además, integrarás componentes de IA en tus flujos: clasificación automática, enriquecimiento de datos y validación inteligente.

Requisito: Python intermedio y familiaridad con SQL y línea de comandos.

Detalles del curso

  • Nivel: Intermedio
  • Duración: 8 semanas (16 horas)
  • Modalidad: Online en vivo + grabaciones
  • Sesiones: 2 horas por semana
  • Proyecto final: Pipeline Airflow + IA en producción
  • Certificado: AIF369 Automatización con Airflow

Contenido del curso

8 módulos con DAGs reales, configuración de entornos y despliegue.

Módulo 1: Introducción a Airflow

  • ¿Qué es Apache Airflow y por qué usarlo?
  • Arquitectura: scheduler, webserver, workers
  • Instalación y setup del entorno
  • Tu primer DAG en 15 minutos

Módulo 2: DAGs y operadores

  • Estructura de un DAG: tasks, dependencies
  • Operadores: Bash, Python, SQL, HTTP
  • Sensores: esperar eventos y archivos
  • XCom: compartir datos entre tareas

Módulo 3: Conexiones y proveedores

  • Connections: bases de datos, APIs, cloud
  • Google Cloud Provider (BigQuery, GCS)
  • AWS Provider (S3, Redshift)
  • Variables y secretos en Airflow

Módulo 4: Pipelines de datos reales

  • Ingestión de datos desde APIs
  • Transformación con pandas y SQL
  • Carga a data warehouse
  • Caso: ETL completo con datos reales

Módulo 5: Airflow + IA

  • Integrar modelos de ML en DAGs
  • Clasificación automática de datos
  • Enriquecimiento con IA generativa
  • Validación inteligente de calidad de datos

Módulo 6: Testing y calidad

  • Testeo de DAGs con pytest
  • Data quality checks integrados
  • Alertas y notificaciones
  • Debugging de DAGs en producción

Módulo 7: Deploy y producción

  • Docker + Airflow: setup productivo
  • Cloud Composer (GCP) / MWAA (AWS)
  • CI/CD para DAGs
  • Monitoreo y scaling

Módulo 8: Proyecto final

  • Pipeline end-to-end con Airflow
  • Componente de IA integrado
  • Desplegado en nube real
  • Presentación y feedback personalizado
