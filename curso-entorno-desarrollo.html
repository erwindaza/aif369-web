🧠 ¿De qué trata este curso?

Configura tu máquina para trabajar como AI Engineer desde día 1. Este curso te guía a través de las herramientas profesionales que todos los desarrolladores de IA usan.

No es solo instalación. Es un flujo de trabajo profesional: cómo mantener código limpio, seguro y reproducible mientras trabajas con APIs de IA.

8 módulos de herramientas profesionales que necesitas como AI Engineer.

  • Instalar Python 3.11+
  • Primeros comandos en terminal
  • Tu primer programa Python
  • Ejecutar y debuggear código

Módulo 2: VS Code — Professional Editor

  • Instalar VS Code
  • Extensiones esenciales (Python, Pylance, Git Graph)

Módulo 3: Git & GitHub

  • Control de versiones profesional
  • SSH keys y autenticación segura
  • Tu primer repositorio en GitHub
  • Flujo: commit, push, pull requests

Módulo 4: Python con uv

  • Gestor de paquetes moderno (uv)
  • Entornos aislados y reproducibles
  • pyproject.toml: configuración profesional
  • Dependencias sin conflictos

Módulo 5: Secretos & Seguridad

  • Variables de entorno (.env)

Módulo 6: Docker

  • Conceptos de containerización
  • Instalar Docker Desktop
  • Tu primer Dockerfile
  • Build & run containers

Módulo 7: AI APIs

  • Generar API keys de forma segura

Módulo 8: AI Agents

  • Qué es un agente de IA
